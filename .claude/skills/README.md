# Skills

Agent skills available in this repo.

## ponytail

The [ponytail](https://github.com/DietrichGebert/ponytail) code-minimalism
system (MIT), vendored verbatim. Six skills — `ponytail`, `ponytail-review`,
`ponytail-audit`, `ponytail-debt`, `ponytail-gain`, `ponytail-help` — that push
agents toward the laziest solution that actually works: YAGNI, stdlib first, no
unrequested abstractions. Invoke with `/ponytail` (or `/ponytail lite|full|ultra`).

This is the house style for bunmaska: zero runtime deps, zero compiled native
code, and a framework held together by *less* code, not more. See
`.admin/AGENTS.md` for the full contribution rules it complements.
